## Commands for GDB with C++:

|           COMMANDS            |              EXPLANATION                    |                 TIPS                |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 01.       start               |  Sets a temporary breakpoint on main() and  |  The start command should only be   |
|                               |  starts executing a program under GDB       |  used if you want to debug a new    |
|                               |                                             |  instance of the program.           |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 02.       list                |  Prints few string from breakpoint          |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 03.       CTRL + X + A        |  Opens visualisation of code                |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 04.       next                |  Goes to & Performs next line of code       |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 05.       CTRL + L            |  Refreshes visualisation of code            |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 06.       q                   |  Exit                                       |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 07.       CTRL + X + 2        |  After CTRL + X + A use  -->  CTRL + X + 2  |                                     |
|                               |  It shows memory with Visual Mode           |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 08.       tui reg float       |  Register in Memory Visulisation            |                                     |
|                               |  -> to float                                |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 09.       CTRL + P /          |  Give previous / next Command in            |                                     |
|           CTRL + N            |  visualisation mode                         |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 10.       python {some py     | Give chance to create little python commands|                                     |
|                      command} | Bcs GDB has py interpreter in               |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 11.       b {name where       | Set breakpoint to some place                |                                     |
|                      break}   |                                             |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 12.       g++ -ggdb3          | New version of Debugger                     |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 13.       set history save on |  CHECK                                      |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 14.       set print pretty on |  CHECK                                      |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 15.       set pagination off  |  CHECK                                      |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 16.       set confirm off     |  CHECK                                      |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
| 17.       watch foo if        |  WatchPoint.                                |                                     |
|                   foo > 10    |                                             |                                     |
|-------------------------------+---------------------------------------------+-------------------------------------|
